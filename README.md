# ml-course


請從清單中找到本週要用的程式碼範例，並依照以下步驟操作：

1. 點選本週的範例。  
2. 點選畫面上方的 **「Open in Colab」** 按鈕。  
3. 開啟後，請點選左上角的 **「檔案」→「在雲端硬碟中儲存副本」**，即可將程式碼儲存到自己的 Google 雲端硬碟中進行編輯與執行。

請先登入 Google 帳號，才能儲存副本與執行程式碼。
